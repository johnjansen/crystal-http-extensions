# http_redirects

Extensions to HTTP::Client enabling redirect follow with cookie replay

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  http_redirects:
    github: johnjansen/crystal-http-extensions
```

## Usage

```crystal
require "http_redirects"
```
